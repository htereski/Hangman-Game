# Rodando localmente

## Clone o projeto

```bash
  git clone https://github.com/htereski/Hangman-Game/tree/main
```

## Entre no diretório do projeto

```bash
  cd Hangman-Game
```

## Instale as dependências

```bash
  npm install
```
```bash
  composer install
```

## Gere uma "APP_KEY" para o .env

```bash
  php artisan key:generate
```

## Inicie o servidor

```bash
  npm run dev
```
```bash
  php artisan serve
```
